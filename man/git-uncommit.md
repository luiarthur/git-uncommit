% GIT-UNCOMMIT(1) | Git Plugin Manual

NAME
====
**git-uncommit** - Undo the last commit and add, keeping the changes so they can be stashed.

SYNOPSIS
========
| **git-uncommit**

DESCRIPTION
===========
Undoes the last `commit` and `add`. This way, you can potentially stash away changes.

Options
=======
None.
